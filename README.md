<h1 align="center">
EmojiDex 表情检察官 - 一个元钞银行的拓展包
</h1>


## Tech

- [Nest.js](https://nestjs.com/)
- [TypeScript](https://www.typescriptlang.org)
- [Bolt by Slack](https://slack.dev/bolt)
- PostgreSQL

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
